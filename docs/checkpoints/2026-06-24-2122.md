## Decisions
- Adopt `slotmap` as the engine over hand-building the arena — maturity is a
  safety property; frees effort for the novel concurrent tier (resolved PLAN Q2).
- Standard `SlotMap` as the default backing over `DenseSlotMap` — wins the hot
  lookup + churn paths; loses only iteration (the colder path). Benched.
- Staged 3b: arc-swap RCU (zero unsafe) as the trusted default; epoch+atomics
  (3b-II) only as the heavier, loom-gated option — not because 3b-I was inadequate
  but because the user asked for all tasks.
- loom bounded to `preemption_bound=3` over an exhaustive run — unbounded hung
  combinatorially (45+ min); bounded covers the torn-read it must catch in <1s.
- Honest verdict that the `byte`/GlobalAlloc tier does NOT beat mimalloc and is
  research-only; resocks5 stays on mimalloc.
